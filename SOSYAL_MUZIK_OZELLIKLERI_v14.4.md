## 🎯 Sosyal Özellikler Kullanımı

### Arkadaş Ekleme
1. Arkadaşlar sekmesine git
2. "Arkadaş Ekle" butonuna tıkla
3. Kullanıcı adını gir
4. Arkadaşlık isteği gönder

### Mesajlaşma
1. İlk kullanımda gizlilik politikasını kabul et
2. Mesajlar sekmesine git
3. Arkadaşını seç
4. Mesaj yaz veya müzik paylaş

### Müzik Paylaşma
1. Sohbet sırasında 🎵 butonuna tıkla
2. Paylaşmak istediğin müziği seç
3. Arkadaşın müziği direkt çalabilir

### Müzik İndirme
1. Herhangi bir şarkının yanındaki ⬇️ butonuna tıkla
2. İndirilenler sekmesinden çevrimdışı dinle
3. İndirilen şarkılardan çalma listesi oluştur

## 📋 Teknik Detaylar

### Firebase Koleksiyonları
- `users` - Kullanıcı bilgileri
- `friends` - Arkadaşlık ilişkileri
- `friendRequests` - Arkadaşlık istekleri
- `messages` - Mesajlar
- `downloads` - İndirilen müzikler

### Yeni JavaScript Fonksiyonları
- `sendFriendRequest()` - Arkadaş isteği gönder
- `loadFriends()` - Arkadaşları yükle
- `sendMessage()` - Mesaj gönder
- `shareMusic()` - Müzik paylaş
- `downloadSong()` - Müzik indir
- `changeLanguage()` - Dil değiştir

### CSS Sınıfları
- `.friends-container` - Arkadaş listesi
- `.chat-modal-content` - Sohbet arayüzü
- `.download-item` - İndirilen müzik
- `.language-selector` - Dil seçici
